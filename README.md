# homebrew-mimi

Homebrew tap for [MIMI](https://github.com/marlonxie/MIMI).

```bash
brew tap marlonxie/mimi
brew install --cask mimi
```

After install, launch from Spotlight ("MIMI") or:

```bash
open -a MIMI
```

## Uninstall

```bash
brew uninstall --zap --cask mimi
```

`--zap` clears user data under `~/Library/Application Support/MIMI/`.
